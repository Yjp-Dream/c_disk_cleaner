随着 Windows 系统长期使用，C 盘会积累大量临时文件、浏览器缓存、系统更新残留、缩略图缓存等冗余数据，导致磁盘空间持续压缩、系统运行速度下降。市面上同类工具普遍存在捆绑广告、收费解锁核心功能、界面复杂等问题，普通用户难以高效使用。本项目立项目的是提供一款完全免费、开箱即用、界面简洁美观的 C 盘清理工具，帮助用户一键释放磁盘空间，提升系统运行效率，降低使用门槛。
项目采用 Python 3 + Tkinter 构建桌面 GUI，使用深色主题（Dark UI）配合自定义颜色系统，实现现代化视觉风格。核心清理模块通过 os、shutil 标准库递归遍历并删除系统临时目录、浏览器缓存（Chrome / Edge）、Windows 更新下载缓存、预读取文件及缩略图缓存，并实时统计释放空间。UI 层与清理逻辑通过 threading 多线程解耦，保证界面在清理过程中不卡顿。进度条采用 ttk.Progressbar 自定义样式，日志区域支持颜色标签区分成功与失败状态。最终通过 PyInstaller 打包为单文件 exe，无需安装 Python 环境，双击即可运行。
